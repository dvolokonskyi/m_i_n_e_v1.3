Комбінації клавіш Minecraft Java Edition

Комбінації клавіш дозволяють ефективніше керувати персонажем, будувати, досліджувати світ чи просто веселитися. Ось список гарячих клавіш та кнопок керування у Minecraft Java Edition:

Керування персонажем

  • W — рух вперед
  • A — рух вліво
  • S — рух назад
  • D — рух вправо
  • Space — стрибок
  • Space (двічі) — політ (тільки в креативі)
  • Shift (утримувати) — присідання
  • Ctrl (утримувати) — біг
  • E — відкрити інвентар
  • Ліва кнопка миші — удар / зламати
  • Права кнопка миші — використати предмет / поставити блок

Взаємодія з предметами

  • 1–9 — вибір слота на панелі швидкого доступу
  • Прокручування колесика миші — вибір слота на панелі швидкого доступу
  • Q — викинути предмет із руки
  • Ctrl + Q — викинути весь стак предметів
  • F — перемістити предмет до другої руки (лівої)
  • Колесико миші — дістати блок з інвертарю
  • Shift + ліва кнопка миші — миттєве переміщення предметів між інвентарями
  • Ctrl + Pick Block — копіювати блок разом з даними (NBT) у креативі

Навігація і вигляд

  • Tab — список гравців на сервері
  • F1 — приховати HUD (інтерфейс користувача)
  • F2 — зробити скріншот
  • F3 — показати відладочну інформацію (координати, навантаження, біом і т.д.)
  • F5 — змінити вигляд камери (від першої/третьої особи)
  • F11 — перемикання повноекранного режиму
  • F3 + N — перемикання між попереднім режимом та режимом спостерігача (тільки в креативі)
  • F3 (утримувати) + F4 — перемикання між усіма режимами гри (тільки в креативі)

Спеціальні комбінації F3

  • 0 — відображення мешн детальних графіків
  • 1–9 — відображення більш детальних графіків
  • F3 + A — перезавантажити всі чанки
  • F3 + B — показати рамки зіткнення (hitbox)
  • F3 + D — очистити чат
  • F3 + F — змінити відстань промальовки
  • F3 + G — показати межі чанків
  • F3 + H — додаткова інформація про предмети (ID, міцність)
  • F3 + I — скопіювати інформацію про блок або моба
  • F3 + Q — показує меню допомоги з комбінаціями клавіш у меню відладки (F3)
  • F3 + P — пауза при згортанні гри (вмикати/вимикати)
  • F3 + T — перезавантажити всі текстури і ресурси

Додаткові клавіші

  • Alt (лівий) + ліва кнопка миші — перенести предмет між інвентарями швидко (в залежності від налаштувань сервера/моду)
  • ESC — пауза/головне меню
  • L — відкрити досягнення
  • T — відкрити чат
  • / — відкрити чат з вже введеним символом "/" (початок команди)
  • Tab (поки в чаті) — вибрати та ввести нік гравця на сервері
